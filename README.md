SAMi
=========

## Description

Hi, I'm Sam! Short for Sympathetic AI Messenger bot.

I use Facebook messenger and API.ai, with some Alexa hooks.

## Deployment

Heroku and API.ai, set environment variables, for example.

```shell
heroku config:set VERIFY_TOKEN=3141516
```
